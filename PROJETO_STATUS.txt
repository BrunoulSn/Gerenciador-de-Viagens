🚀 COMO EXECUTAR
═══════════════════════════════════════════════════════════════════════════

1. Abrir no Android Studio
   File → Open → Selecionar MyApplication

2. Sincronizar Gradle
   Gradle sincronizará automaticamente

3. Executar Projeto
   Shift+F10 (Windows) ou ⌘R (Mac)

4. Testar
   ├─ Tela de Login será exibida
   ├─ Clique em "Novo Usuário" para Registro
   ├─ Teste validações deixando campos vazios
   └─ Navegue entre as telas
